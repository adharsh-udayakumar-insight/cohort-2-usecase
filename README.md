# cohort-2-usecase

Files added
- `backup_project.sh`: simple backup helper script
- `generate_workspace.sh`: scaffolds a workspace layout

Quick start
1. Make scripts executable: `chmod +x backup_project.sh generate_workspace.sh`
2. Run the generator: `./generate_workspace.sh`

# Step 1: Make scripts executable
chmod +x backup_all.sh backup_project.sh generate_workspace.sh

# Step 2: Generate sample workspace
./generate_workspace.sh

# Step 3: Backup a single project
./backup_project.sh ./Projects/WebApp

# Step 4: Backup all projects at once
./backup_all.sh
